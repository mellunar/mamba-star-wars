Ao acessar a página de personagem, um número aleatório até 50 é escolhido e então é feito uma chamada para a API do Star Wars para carregar o personagem com mesmo id do número. Se essa chamada for bem sucedida, os dados do personagem são salvos no estado local. Futuramente, se o número escolhido corresponder aos dados de um personagem que já foi salvo, os dados salvos serão carregados e uma nova chamada para a API não será feita.

A página de histórico contém uma lista com os nomes e ids dos personagens já carregados e a data em que cada um foi acessado. Ao clicar em um dos personagens, haverá um redirecionamento para a página de personagens com os dados do escolhido.
